## ESP32 BLE Mesh Sensor Nodes

### This repo is how to:

1. Promote device information
2. Get data DHT22 in GPIO21
3. Send state GHT22

### How to use:

1. Install IDE and ESP-IDF SDK
2. Git clone https://github.com/xd3011/ESP32-OnOff-Nodes
3. Build and flash to board
4. Done
